HTML List Types

Unordered List (ul)

Bulleted list where order doesn't matter.

  • Bread
  • Milk
  • Eggs
  • Fruits
    • Apples
    • Bananas
    • Oranges

Ordered List (ol)

Numbered list where order is important.

  1. Preheat oven to 350°F (180°C)
  2. Mix dry ingredients
  3. Add eggs and butter
  4. Bake for 30 minutes

Ordered list starting from 10:

  1. Tenth item
  2. Eleventh item
